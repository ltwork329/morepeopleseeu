# 让更多人看到你 v1.0

本地短视频自动生产工作台，用于把文案、配音、视频素材、标题字幕和导出记录串成一条可测试的生产流程。

## 功能

- 文案导入：支持复制粘贴和 Excel 上传。
- 声音处理：支持声音克隆、选择声音、生成音频。
- 视频合成：按任务选择音频，匹配素材，预览首帧，调整标题和字幕，再导出视频。
- 素材库：按 `unused`、`fragments`、`used` 管理本地素材，避免已用素材重复参与合成。
- 操作记录：保留关键动作和失败原因，方便回看。

## 本地运行

```bash
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173/
```

本地 API 默认运行在：

```text
http://127.0.0.1:3210/
```

## 素材目录

首次运行后可以在页面里一键创建并扫描素材目录，也可以运行：

```bash
npm run materials:init
npm run scan:materials
```

素材放入：

```text
local_materials/unused
```

生成结果和素材库存属于本地运行数据，默认不会提交到 GitHub。
